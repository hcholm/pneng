# Pneŋ Orthography

[Overview](README.md) | [Orthography](pneng-orthography.md) | [Grammar](pneng-grammar.md) | [Lexicon](pneng-lexicon.md) | [Examples](pneng-examples.md) | [Sample Text](pneng-sample-text.md)

This document defines the orthography for Pneŋ, covering the full core phoneme inventory of General American English.

## 1. Design Principles

1. One phoneme, one grapheme where practical.
2. Fixed digraphs allowed only when they improve readability.
3. Orthography is phonemic, not etymological.
4. Morphology does not trigger spelling changes.
5. No silent letters.
6. No consonant doubling for any reason (etymological or otherwise).

## 2. Forbidden Letters

Never use `q`, `c`, or `x` in any Pneŋ lexical form.

Replacements:

| Avoid | Use instead |
|---|---|
| hard `c`, `q` | `k` |
| soft `c` | `s` |
| `x` /ks/ | `ks` |
| `x` /gz/ | `gz` |

## 4. Vowel Inventory

### 4.1 Monophthongs

| IPA | Pneŋ | Keyword |
|---|---|---|
| /iː/ | `ī` | FLEECE |
| /ɪ/ | `i` | KIT |
| /eɪ/ → see diphthongs | — | FACE |
| /ɛ/ | `e` | DRESS |
| /æ/ | `æ` | TRAP |
| /ɑ/ | `a` | PALM / LOT (merged in GA) |
| /ɔ/ | `o` | THOUGHT (if distinct) |
| /ʊ/ | `u` | FOOT |
| /uː/ | `ū` | GOOSE |
| /ʌ/ | `ä` | STRUT |
| /ə/ | `ə` | schwa (unstressed syllables) |
| /ɝ/ | `ö` | NURSE ("first", "word", "bird") |

Notes:
- `ö` is the standard symbol for the NURSE vowel class throughout Pneŋ.
- Dialects that merge /ɑ/ and /ɔ/ write the merged form as `a`.

## 5. Stress and Reduction

1. Stress is not marked in ordinary running text.
2. Unstressed reduced vowels are written `ə` where clarity requires it.
3. High-frequency function words may lexicalize with reduced vowels.

## 6. Allophony (Not Written)

The following are phonetic, not phonemic, and are not written:

1. Aspirated vs. unaspirated stops.
2. Flap [ɾ] for /t d/ between vowels.
3. Dark [ɫ] vs. light [l].
4. Nasalized vowels before nasals.
5. Voicing assimilation in clusters.

## 7. Conversion from English Spellings

When importing English vocabulary:

| English | Pneŋ rule |
|---|---|
| `th` (voiced) | `ð` |
| `th` (voiceless) | `þ` |
| `sh` | `š` |
| `ch` | `č` |
| `j`, `dge`, `ge` | `ǰ` |
| `hard c`, `k`, `qu` | `k` |
| `soft c` | `s` |
| `x` /ks/ | `ks` |
| `x` /gz/ | `gz` |
| `y` (consonant) | `j` |
| `ng` /ŋ/ | `ŋ` |
| `ng` /ŋg/ | `ŋg` |
| double consonant | single consonant |

## 9. Compliance Checklist

- [ ] No `q`, `c`, or `x` in lexical forms.
- [ ] No consonant doubling.
- [ ] `ŋ` used (not `ng`) for /ŋ/.
- [ ] `j` used (not `y`) for consonantal /j/.
- [ ] `ö` used for NURSE vowel.
- [ ] TH distinguished as `þ` (voiceless) vs `ð` (voiced).
